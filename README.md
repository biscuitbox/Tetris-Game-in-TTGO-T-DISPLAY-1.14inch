# Tetris-Game-in-TTGO-T-DISPLAY-1.14inch
Tetris in  TTGO T-DISPLAY 1.14inch

1. 딥슬립모드
  * 1분간 입력이 없으면 딥슬립모드(절전모드)로 진입하면서 화면 꺼짐.
  * 딥슬립모드에서 왼쪽 버튼 누르면 부팅되면서 실행됨.

2. 부팅 → 로고 화면(로고 이미지만)
  * 버튼 안 누르면 그대로 대기
  * 1분 동안 아무 입력 없으면 딥슬립

3. 버튼 누르면 → “첫 화면(타이틀/설명 화면)”
  * “TETRIS”, “버튼을 눌러 시작” 같은 문구 표시
  * 여기서도 버튼 누르기 전까지는 게임 화면으로 안 넘어감
  * 역시 1분 동안 무입력이면 딥슬립

4. 첫 화면에서 버튼 누르면 → 실제 게임 화면 + 블록 생성
  * 이때도 첫 좌/우 버튼 입력 전까지는 블록이 안 떨어짐
  * 이후부터는 기존 테트리스처럼 자동 낙하
